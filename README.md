# AI Job Hunter

AI-powered Career Intelligence Platform for Job Seekers.

## Features

### Resume Intelligence

* Resume Parsing
* ATS Resume Analysis
* Resume Optimization
* Skill Extraction
* Skill Gap Analysis

### Job Matching

* AI Job Matching
* Match Score Calculation
* Job Collection from APIs
* Job Recommendations

### Career Development

* AI Career Coach
* Learning Roadmaps
* Certification Recommendations
* Career Progression Guidance

### Application Tracking

* Application Tracker
* Application Status Updates
* Saved Jobs
* Watchlist

### Recruiter CRM

* Recruiter Tracking
* Contact Management
* Follow-up Tracking

### LinkedIn Intelligence

* LinkedIn Headline Generator
* LinkedIn About Generator
* Profile Improvement Suggestions

### Authentication

* User Registration
* Login & Logout
* Multi-User Support
* User-Specific Data Isolation

## Technology Stack

* Python
* Streamlit
* SQLite
* Sentence Transformers
* Scikit-Learn
* Pandas
* Plotly

## Installation

```bash
git clone <repository-url>
cd AI-Job-Hunter
pip install -r requirements.txt
streamlit run dashboard.py
```
